# Regália Sörház – Webshop Frontend és UI/UX Tervezés

**Készségek és technológiák:** 
`HTML5` | `CSS3` | `Bootstrap 5` | `UI/UX Drótvázak (Wireframe)` | `W3C Szabványok` | `Akadálymentesítés (a11y)`

Egy modern kézműves sörfőzde webshop frontendjének teljes körű megtervezése és lefejlesztése. A folyamat a strukturális drótvázak (wireframe-ek) elkészítésével indult, amelyet a Bootstrap keretrendszerre épülő, "mobile-first" szemléletű kódolás követett.

## Főbb fókuszpontok a fejlesztés során

* **Precíz UI/UX átültetés:** A megtervezett drótvázak reszponzív, felhasználóbarát felületté alakítása.
* **Szigorú W3C megfelelés:** Szemantikusan tiszta, validált HTML5 kódstruktúra és logikus oldalarchitektúra kialakítása.
* **Kiemelt akadálymentesítés (Accessibility):** A WAI-ARIA sztenderdek betartása, képernyőolvasók (screen reader) támogatása, "Ugrás a fő tartalomra" funkció, valamint egyedi akadálymentes nézet biztosítása.
